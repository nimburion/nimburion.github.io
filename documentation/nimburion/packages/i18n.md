# pkg/i18n

Internationalization support.

## Load Translations

```go
import "github.com/nimburion/nimburion/pkg/i18n"

i18n.Load("locales")

// locales/en.json
// locales/it.json
```

## Translate

```go
msg := i18n.T("welcome_message", "en")
msg := i18n.T("welcome_message", "it")

// With parameters
msg := i18n.T("greeting", "en", map[string]string{
    "name": "Alice",
})
```
